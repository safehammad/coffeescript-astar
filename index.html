    CoffeeScript A* Maze Solver

    This is an implementation of the A* algorithm in CoffeeScript implemented as a maze solver. The maze solving algorithm can be found in astar.coffee. Although this particular implementation is specific to solving basic mazes, the A* implementation should be easy to generalise.

    Usage

    The maze solver in astar.coffee expects a maze as a simple string describing the location of the walls and the start and finish points. The solver returns the best path from start to finish as an array of coordinates.

    For example:

    # Load the astar script
{astar} = require('./astar')

# Define a maze as a string where:
#
#  '@' = start point
#  'F' = end point
#  '*' = wall
maze ='''*********
         *       *
         *   *   *
         * @ * F *
         *********'''

# Create the solver
solver = new astar.MazeSolver maze

# Now get the solved route!
route = solver.solve()

# The route is a series of [row, column] pairs from start to finish
console.log route

# Output:
# [ [ 3, 2 ],
#   [ 3, 3 ],
#   [ 2, 3 ],
#   [ 1, 3 ],
#   [ 1, 4 ],
#   [ 1, 5 ],
#   [ 2, 5 ],
#   [ 2, 6 ],
#   [ 3, 6 ] ]

    Demo

    This interactive demo (also found in index.html) visualises the maze via SVG using the Raphael JavaScript library (included).

    • Click the Next and Previous buttons to move the trail forwards and backwards.
    • Click the Restart button to move the trail back to the start point.
    • Click anywhere in the maze to change the location of the start point.
