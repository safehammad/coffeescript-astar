    CoffeeScript A* Maze Solver

    This is an implementation of the A* algorithm in CoffeeScript implemented as a maze solver. The maze solving algorithm can be found in astar.coffee. Although this particular implementation is specific to solving basic mazes, the A* implementation should be easy to generalise.

    A sample implementation is provided in index.html (see below) which visualises the maze using SVG via the Raphael JavaScript library (included). Click the Next and Previous buttons to move the trail forwards and backwards. Click the Restart button to move the trail back to the start point. The location of the start point can be altered by clicking anywhere on the maze.

    Enjoy...

    How do I use this?

    
# Load the astar script:
{astar} = require('./astar')

# Define a maze as a string where:
#
#  '@' = start point
#  'F' = end point
#  '*' = wall
maze ='''*********
         *       *
         *   *   *
         * @ * F *
         *********'''

# Create the solver
solver = new astar.MazeSolver maze

# Get the solved route!
route = solver.solve()

# The route is a series of [row, column] pairs from start to finish
console.log route

# Output:
# [ [ 3, 2 ],
#   [ 3, 3 ],
#   [ 2, 3 ],
#   [ 1, 3 ],
#   [ 1, 4 ],
#   [ 1, 5 ],
#   [ 2, 5 ],
#   [ 2, 6 ],
#   [ 3, 6 ] ]
